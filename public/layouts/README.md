# Resume Layouts

Place your layout preview images and JSON schemas here.

## File Structure

- `layout-modern.png` - Preview image for modern layout
- `layout-classic.png` - Preview image for classic layout
- `layout-creative.png` - Preview image for creative layout

## Adding New Layouts

1. Add preview image: `layout-{id}.png`
2. Update `app/lib/layouts.ts` with new layout definition
3. Layout will automatically appear in the selector
